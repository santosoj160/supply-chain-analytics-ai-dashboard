# Supply Chain Analytics & AI Forecasting Dashboard

## 📌 Project Overview

This project presents an end-to-end Supply Chain Analytics solution built using Python, PostgreSQL, SQL, and Tableau. The project analyzes sales performance, profitability, logistics operations, and predictive business trends using real-world supply chain data from DataCo Global.

The dashboard provides interactive business intelligence insights and AI-powered revenue forecasting to support strategic decision-making.

---

# 🎯 Business Objectives

The project aims to:

* Analyze sales and profitability performance
* Monitor logistics and delivery operations
* Identify top-performing products and customer segments
* Detect operational risks and shipping delays
* Forecast future revenue trends using AI forecasting models
* Build interactive executive dashboards for business decision-making
